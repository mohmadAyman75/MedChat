# MedChat Front End

واجهة عربية من اليمين إلى اليسار لتجربة شات صحي محلية.

## الملفات

- `index.html`: هيكل الصفحة والقوائم.
- `styles.css`: تصميم الشات، السايدبار، الكومبوزر، والقوائم المتجاوبة.
- `script.js`: إدارة المحادثات، الرسائل، التخزين المحلي، القوائم، والردود التجريبية.

## التشغيل

افتح `index.html` مباشرة في المتصفح.
